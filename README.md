# k-Notes
Android Notes App Using Kotlin

## What's inside
- Kotlin
- SQLiteDatabase
- LinearLayout
- CoordinatorLayout
- ListView
- SearchView
- FloatingActionButton
- ScrollView
- AlertDialog
